- ✅ Детальная информация о товаре
- ✅ Система корзины

2. Просмотреть товары в категории

5. Просмотреть корзину
6. Удалить товаров из корзины
7. Оформить заказ
8. Ввести данные (имя, фамилия, телефон, адрес)
9. Подтвердить заказ
10. Получить чек
11. Просмотреть историю заказов
12. Посмотреть детали заказа
13. Посмотреть свой профиль

## 🗄️ БАЗА ДАННЫХ

### 7 Таблиц:
1. **categories** - категории товаров
2. **products** - товары с ценой и остатком
3. **users** - пользователи Telegram
4. **carts** - корзины
5. **cart_items** - товары в корзинах
6. **orders** - заказы
7. **order_items** - товары в заказах

### Связи:
- Category → Products (1:many)
- User → Carts (1:many)
- User → Orders (1:many)
- Cart → CartItems (1:many)
- Product → CartItems (1:many)
- Order → OrderItems (1:many)
- Product → OrderItems (1:many)

2. Категория → Выбор товара

4. Корзина → Оформить
5. Оформление → Данные (4 шага)
6. Подтверждение → Чек

## 💡 КЛЮЧЕВЫЕ МОМЕНТЫ

### 1. Выбор БД в .env

```env
# SQLite (разработка)
DB_TYPE=sqlite

# MySQL (продакшен) ← РЕКОМЕНДУЕТСЯ
DB_TYPE=mysql

# PostgreSQL (масштабирование)
DB_TYPE=postgresql
```

### 2. MySQL параметры в .env

```env
DB_HOST=localhost       # Адрес MySQL
DB_PORT=3306           # Порт (обычно 3306)
DB_USER=root           # Пользователь
DB_PASSWORD=password   # Пароль
DB_NAME=shop_db        # Имя БД
```

### 3. Автоматическое подключение

```python
# db/database.py читает .env и подключается автоматически
db_type = os.getenv("DB_TYPE")  # mysql
if db_type == "mysql":
    DATABASE_URL = f"mysql+aiomysql://{user}:{password}@{host}/{db}"
    # Подключается к вашей MySQL!
```

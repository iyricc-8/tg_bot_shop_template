# ⚡ БЫСТРЫЙ СТАРТ - 5 МИНУТ

## 🚀 Запустите магазин за 5 шагов!

### 1️⃣ Установить Python пакеты (1 сек)

```bash
pip install -r requirements.txt
```

### 2️⃣ Создать .env файл (1 сек)

Скопируйте и отредактируйте:

```bash
cp .env.example .env
```

Затем отредактируйте файл .env:
```
BOT_TOKEN=YOUR_TOKEN_HERE
ADMINS=YOUR_ID_HERE
```

### 3️⃣ Инициализировать БД (1 сек)

```bash
python init_shop_db.py
```

Должно появиться:
```
✅ БД инициализирована успешно!
✅ Создано категорий: 5
✅ Создано товаров: 15
```

### 4️⃣ Получить BOT_TOKEN (2 мин)

1. Открыть Telegram
2. Найти @BotFather
3. Отправить `/newbot`
4. Дать имя боту (например: "My Shop Bot")
5. Дать юзернейм (должен быть уникальный)
6. Скопировать токен
7. Вставить в .env

### 5️⃣ Запустить магазин (1 сек)

```bash
python main.py
```

Должно появиться:
```
Starting polling...
```

## 🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

| Проблема | Решение |
|----------|---------|
| `ModuleNotFoundError` | `pip install -r requirements.txt` |
| `BOT_TOKEN not found` | Проверьте .env файл |
| `Database error` | `python init_shop_db.py` |
| У бота нет ответа | Перезагрузите `python main.py` |

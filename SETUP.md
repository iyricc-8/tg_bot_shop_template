# 🚀 ИНСТРУКЦИИ ПО ЗАПУСКУ МАГАЗИНА

## 📋 Требования

- Python 3.8+
- pip

## ✅ Пошаговая установка

### 1️⃣ Установка зависимостей

```bash
# Установить все требуемые пакеты
pip install -r requirements.txt
```

### 2️⃣ Конфигурация

```bash
# Скопировать пример файла
cp .env.example .env

# Отредактировать .env и добавить:
# - BOT_TOKEN (получить от @BotFather в Telegram)
# - ADMINS (ваш ID в Telegram)
```

#### Как получить BOT_TOKEN:
1. Открыть Telegram
2. Найти бота @BotFather
3. Отправить команду `/newbot`
4. Следовать инструкциям
5. Скопировать токен в .env

#### Как получить ваш Telegram ID:
1. Найти бота @userinfobot
2. Отправить любое сообщение
3. Бот вернет ваш ID
4. Вставить в ADMINS в .env

### 3️⃣ Инициализация базы данных

```bash
# Это создаст БД с примерами товаров
python init_shop_db.py
```

Вы должны увидеть:
```
✅ БД инициализирована успешно!
✅ Создано категорий: 5
✅ Создано товаров: 15
```

### 4️⃣ Запуск бота

```bash
# Запустить бота
python main.py
```

Вы должны увидеть:
```
Starting polling...
```

## 🎮 Использование

1. Открыть Telegram
2. Найти вашего бота (по имени, которое вы дали @BotFather)
3. Отправить `/start`
4. Использовать меню для покупок

## 🔧 Управление магазином

### Добавить товар вручную

```python
from db.database import AsyncSessionLocal, init_db
from db.repositories import ProductRepository
import asyncio

async def add_product():
    await init_db()
    async with AsyncSessionLocal() as session:
        await ProductRepository.create(
            session,
            name_uz="Мой товар",
            price=50000,
            category_id=1,
            description_uz="Описание",
            stock=20
        )

asyncio.run(add_product())
```

### Админская консоль

```bash
# Консоль для управления магазином
python admin_console.py
```

## 🐛 Решение проблем

### Ошибка: "No module named 'aiogram'"

```bash
pip install -r requirements.txt
```

### Ошибка: "BOT_TOKEN не найден"

Убедитесь что:
1. Файл `.env` существует
2. В .env установлен `BOT_TOKEN`
3. Он не имеет пробелов перед/после

### Ошибка: База данных не создается

```bash
# Удалить старую БД
rm bot_shop.db

# Пересоздать
python init_shop_db.py
```

### Бот не отвечает

1. Проверьте интернет соединение
2. Убедитесь что токен правильный
3. Перезагрузите бота

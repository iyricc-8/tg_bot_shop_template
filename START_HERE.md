## 🚀 БЫСТРЫЙ СТАРТ

### 3 простые команды:

1. **Установить зависимости:**
```bash
pip install -r requirements.txt
```

2. **Инициализировать БД:**
```bash
python init_shop_db.py
```

3. **Запустить:**
```bash
python main.py
```

**ВСЕ!** Ваш магазин работает! 🎊

## ❓ ЧАСТЫЕ ВОПРОСЫ

### Где найти BOT_TOKEN?
Найдите @BotFather в Telegram, отправьте /newbot и следуйте инструкциям

### Как добавить свои товары?
Отредактируйте `init_shop_db.py` и запустите: `python init_shop_db.py`

### Как удалить примеры?
Удалите `bot_shop.db` и создайте новую БД: `python init_shop_db.py`

### Как использовать другую БД?
Измените DATABASE_URL в `db/database.py`

### Как добавить админа?
Отредактируйте ADMINS в .env файле
